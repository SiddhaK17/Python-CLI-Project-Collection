# 🎯 Number Guessing CLI Game

> _A captivating terminal-based number guessing game that fuses logic, probability, and user interactivity — built with clean Python architecture._

---

## 🧩 Description

The **Number Guessing CLI Game** is a console based Python application that challenges users to deduce a randomly chosen number between 1 and 100. Players are given a fixed number of attempts based on the chosen difficulty level and are guided by intelligent feedback after each guess. With a minimalistic interface enhanced by ASCII art and modular Python functions, this project balances educational value with entertainment.

Whether you're sharpening your coding skills or looking for a lightweight interactive Python experience, this game delivers an ideal blend of logical reasoning, structured code design, and user engagement.

## 📘 Overview

This project simulates the classic number guessing game with an engaging user flow and strategic gameplay. It's structured to showcase **clean function driven programming**, **conditional logic**, and **user feedback loops** while integrating optional enhancements like ASCII visuals and difficulty toggles.

## 🕹️ Gameplay Mechanics

- 🔢 **Game Objective**: Guess a number between `1` and `100` generated by the program.
- 🧠 **Difficulty Levels**:
  - `Easy`: 10 attempts
  - `Hard`: 5 attempts
- 💡 **Smart Feedback System**:
  - Too high? Too low? Get instant directional hints.
  - No feedback on the correct number unless you run out of turns — keeps suspense intact.
- 🏁 **End Conditions**:
  - Game ends when the correct number is guessed or attempts run out.
  - Correct number is only revealed if the user loses — preserving challenge.
